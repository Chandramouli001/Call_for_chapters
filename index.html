About the Book

Tiny Machine Learning (TinyML) enables machine learning models to run directly on ultra-low power microcontrollers and IoT devices, enabling intelligent edge computing.

This edited book presents research contributions and real-world applications of TinyML across domains such as healthcare, smart agriculture, wearable devices, and industrial IoT.

Aim & Scope

The book aims to present recent advancements in Tiny Machine Learning and its integration with IoT systems. TinyML enables machine learning models to run on resource-constrained devices such as microcontrollers and embedded systems, enabling efficient edge intelligence with minimal power consumption.

This edited volume provides a platform for researchers, academicians, and industry professionals to share innovative architectures, deployment strategies, and real-world implementations of TinyML-enabled IoT systems.

Key Topics

  • Introduction to TinyML: Concepts, Motivation, and Use Cases
  • Edge Intelligence and TinyML for Resource-Constrained IoT Devices
  • TinyML for Smart Healthcare: Remote Monitoring and Assistive Systems
  • TinyML in Wearables and Body-Area Networks
  • TinyML for Industrial IoT, Smart Agriculture and Home Automation
  • Neural Network Architectures for Ultra-Low-Power Devices
  • Model Optimization for TinyML: Quantization, Pruning, and Compression
  • Energy-Efficient Learning and Inference Techniques
  • Deployment Challenges and Design Strategies for TinyML
  • Security, Privacy, and Trust in TinyML-Enabled IoT Systems
